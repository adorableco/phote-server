## 🌳 Branch

🌱 git branch 전략

`main branch` : 배포 단위 branch

`develop branch` : 주요 개발 branch, main merge 전 거치는 branch

`name/swm-number branch`: 각자 개발 branch, number는 Linear issue number

### 🌳 branch 구조

```jsx
- main
- develop
   └── name/swm-number
```
