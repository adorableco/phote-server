## 🌳 Branch

🌱 git branch 전략

`release branch` : 배포 단위 branch

`develop branch` : 주요 개발 branch, release merge 전 거치는 branch

`name/swm-number branch`: 각자 개발 branch, number는 Linear issue number

### 🌳 branch 구조

```jsx
- release
- develop
   └── name/swm-number
```
